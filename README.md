# Stock-Data-Extraction-and-Visualization
This project focuses on extracting, cleaning, and visualizing stock market data using Python. The analysis includes retrieving historical stock prices with yfinance, scraping company revenue data from web pages using BeautifulSoup, and creating interactive visualizations with matplotlib.

**Project Objectives**
Extract historical stock data for Tesla and GameStop using yfinance
Scrape quarterly revenue data from web sources
Clean and preprocess financial datasets using pandas
Visualize stock prices and revenue trends
Build reusable graphing functions for financial analysis

**Technologies Used**
Python
Pandas
BeautifulSoup (bs4)
Requests
Matplotlib
Jupyter Notebook
Dataset & Analysis

**The project analyzes:**
Tesla (TSLA)
GameStop (GME)

**For each company:**
Historical stock prices are collected
Revenue data is extracted and cleaned
Graphs are generated to compare stock performance and revenue trends over time
